# Project Brief

Camphora is a modern, accessible UI component library built with React and styled using vanilla-extract CSS-in-TS. It provides a comprehensive set of reusable components for building web applications, focusing on simplicity, customization, and adherence to web standards.

## Core Goals
- Provide high-quality, accessible UI components
- Enable easy customization through design tokens
- Support responsive design out of the box
- Maintain consistency across components
- Offer excellent developer experience with TypeScript support

## Scope
The library includes components for common UI patterns such as buttons, forms, navigation, overlays, and data display. It is designed as a monorepo with separate packages for React components and styling.
