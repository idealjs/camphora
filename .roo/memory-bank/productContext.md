# Product Context

## Why Camphora Exists
Camphora addresses the need for a lightweight, customizable UI component library that bridges the gap between design systems and developer productivity. It solves common problems like inconsistent styling, accessibility issues, and time-consuming component development.

## Problems Solved
- **Inconsistent UI**: Provides a unified design language
- **Accessibility**: Built-in ARIA support and keyboard navigation
- **Customization**: Flexible theming through design tokens
- **Performance**: Optimized CSS and minimal bundle size
- **Developer Experience**: TypeScript support and clear APIs

## User Experience Goals
- Intuitive component APIs that follow React conventions
- Responsive components that work across devices
- Smooth animations and transitions
- Clear visual feedback for interactions
- Comprehensive documentation with live examples

## Target Audience
- Frontend developers building web applications
- Design system maintainers
- Teams looking for a balance between customization and ease of use
