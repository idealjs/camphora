# Modal — Playwright 测试说明

- 测试目标：验证模态框的打开/关闭流程、遮罩与焦点管理（若 JS 实现），并做视觉快照。
- 推荐测试用例：
  - 打开/关闭快照：点击打开与关闭模态，截图两种状态。
  - 焦点陷阱：打开模态后断言聚焦在模态内第一个可聚焦元素，Tab 循环不应离开模态（如果实现焦点陷阱）。
  - 遮罩点击：若实现遮罩点击关闭，模拟点击并断言模态关闭。
  - ESC 关闭：按 ESC 键断言模态关闭（如有实现）。
- 建议测试文件：`apps/website/tests/modal.spec.ts`
- 注意：如果文档仅提供 CSS-only 示例，重点做视觉快照并在注释中记录可访问性限制。
